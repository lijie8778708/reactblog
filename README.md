## A full stack project basic on react js and spring boot

### Tech

`Front end` : 
* material-ui
* axios
* react
* redux
* react-markdown
* for-editor

`Back end` : 
* mybatis plus
* springboot
* mysql
* shiro
* redis
* jwt
